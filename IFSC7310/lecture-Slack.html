Chat Rooms, Forums, and the Slack Example


What is a chat room?
A themed thread of text messages with group access
A set of related persistent chat rooms are a "forum"
  • Example: Ars Technica has forums that each have chat rooms
More on Forums
  • An "internet forum"
  • Traditional name: message board
  • A collection of related persistent chat rooms
  • Persistent?
  • Themed?
  • Thread?


















  • Persistent: stored for future access
  • Themed: sharing a topic or other commonality
  • Thread: sequence of messages
