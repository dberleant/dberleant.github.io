Syntax to "bake" a cookie:
document.cookie = "attribute = value"; //Simple case without options
Examples:
JavaScript code:
document.cookie = "course=IFSC 30003"; //Session-only cookie
Cookie specified by the code:
course=IFSC 3300
JavaScript code:
document.cookie = "course=IFSC 30003; expires=Thu, 01 Jan 2026 00:00:00 UTC";
Cookie specified by the code:
course=IFSC 30003; expires=Thu, 01 Jan 2026 00:00:00 UTC
To delete it:
document.cookie = "course=; expires=Thu, 01 Jan 1970 00:00:00 UTC"; Now bake your own cookie!

Example code to "taste" a cookie:
document.getElementById('printHere').innerHTML+=document.cookie;
