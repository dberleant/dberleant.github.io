Syntax to "bake" a cookie:
    document.cookie = "attribute = value"; //Simple case without options
Examples:
1. JavaScript code for temporary cookie:
    document.cookie = "course=IFSC 30003"; //Session-only cookie
    Cookie specified by the code:
    course=IFSC 3300
2. JavaScript code to persist a cookie:
    document.cookie = "course=IFSC 30003; expires=Thu, 01 Jan 2028 00:00:00 UTC";
    Cookie specified by the code:
    course=IFSC 30003; expires=Thu, 01 Jan 2028 00:00:00 UTC
3. Javascript code to delete a cookie:
    document.cookie = "course=; expires=Thu, 01 Jan 1970 00:00:00 UTC";
    This deletes the course cookie

Example code to "taste" a cookie:
    document.getElementById('printHere').innerHTML += document.cookie;
