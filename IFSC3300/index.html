    Computer, textbooks and related resources:

    • Please bring your laptop to class for class days when there will be hands-on exercises. There are also a few machines in the back of the classroom.
    • The JavaScript programming book that was originally considered was $150 (for a paperback!) and is not necessary. Instead, a modern and better alternative is on-line, interactive resources. We will use w3schools.com in this course. There are also other online resources with JavaScript sections, like The Odin Project, Khan Academy (favorably reviewed in an article), tutorialspoint.com, udemy.com, teamtreehouse.com, guru99.com, codecademy.com, codepen.io, jsbin.com, codeanywhere.com, javascriptkit.com and code.visualcode.com. These are of widely varying usefulness. Many have the added benefit of having some free learning resources (at least for now, though these sites may update their business models at some point by charging).
    • Free textbooks (for this and many other subjects) are also available, for example from http://www.textbookrevolution.org. One of their JavaScript textbooks is Eloquent JavaScript, available at http://eloquentjavascript.net. Another free textbook is at javascript.info. Still another is at https://www.learn-js.org/.
    • Language reference material is available at https://developer.mozilla.org/en-US/docs/Web/JavaScript. The official specification (as of this writing) is at https://www.ecma-international.org/publications/standards/Ecma-262.htm and - true fact - the official name of the language is not JavaScript, but ECMAscript. JavaScript is technically Oracle's brand name, and calling ECMAscript "JavaScript" is a lot like calling cola "Coke" or "Pepsi."

    Student learning objectives:

    • At the end of this course students will be able to do client-side programming in JavaScript.
    • At the end of this course students will be able to engage in independent professional development in JavaScript and other languages using online learning websites.

    Catalog description: IFSC 3300 Internet Applications
    Prerequisite: IFSC 1310 or equivalent, or consent of Instructor. A hands-on course focusing on the technologies and concepts for creating dynamic and interactive web sites with a special emphasis on client-side technologies. Topics will cover techniques such as how to build efficient and dynamic interactive user interfaces, how to interface with data using standardized, portable formats, how to store/validate data and how to make data more accessible to other applications. Three hours lecture. Three credit hours.
